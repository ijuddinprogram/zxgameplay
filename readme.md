# ZXGameplay2 — Halaman Sosmed Gaming

Halaman landing page sosial media untuk kreator gaming **ZXGameplay2**, dibuat dengan HTML, CSS, dan JavaScript murni. Menampilkan link ke channel YouTube dan TikTok, statistik channel, serta desain bertema gaming dark futuristic.

## Preview

![ZXGameplay2 Preview](https://placehold.co/800x500/0a0a0f/ffffff?text=ZXGameplay2+Preview)

## Fitur

- **Avatar animasi "ZX"** — efek pulse glow merah-ungu.
- **Tombol CTA YouTube & TikTok** — link langsung ke channel `@ZXGameplay2`.
- **Statistik channel** — jumlah video, subscriber, dan game favorit (MLBB).
- **Dark futuristic theme** — gradien merah + cyan, glassmorphism, hover animasi.
- **Responsive** — tampil optimal di desktop maupun mobile.
- **SVG icon native** — tanpa dependensi eksternal.

## Struktur File

zxgameplay2.html     ← file utama (buka di browser)
README.md            ← dokumentasi ini

## Cara Menjalankan

1. Clone repository ini atau download file `zxgameplay2.html`.
2. Buka file tersebut di browser apa pun (Chrome, Firefox, Edge, Safari).
3. Halaman langsung tampil — tidak perlu server atau build tool.

# Atau langsung buka dari terminal:
start zxgameplay2.html       # Windows
open zxgameplay2.html        # macOS
xdg-open zxgameplay2.html    # Linux
